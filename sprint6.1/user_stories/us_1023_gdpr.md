# US_1023 GDPR

## User story

As a user, \
I want to see a statement of how my data is protected in the app.

## Acceptance criteria

*Dirty road:* \
New button "GDPR" in main menu is displayed. \
When clicked on it links us to the GDPR official website main page.

## Tests
### Scenario: User clicks many times
**Given** I am a user who wants to go to GDPR page
**When** I click multiple times on the button linked to GDPR website.
**Then** A new tab with GDPR website should be opened once.
